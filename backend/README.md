# 🚀 TaskFlow Pro

TaskFlow Pro is a full-stack project management and collaboration platform inspired by tools like Trello and Asana. It allows teams to manage projects, organize tasks, track progress, collaborate efficiently, and monitor productivity through an interactive dashboard.

---

## 📌 Features

### Authentication
- User Registration & Login
- JWT Authentication
- Password Hashing (bcrypt)
- Role-Based Access Control
- Profile Management

### Project Management
- Create, Update and Delete Projects
- Archive & Restore Projects
- Add or Remove Team Members
- Activity Tracking

### Task Management
- Create, Edit and Delete Tasks
- Kanban Workflow (To Do → In Progress → Completed)
- Task Priorities & Due Dates
- Labels and Tags
- Checklist Support
- File Attachments

### Collaboration
- Task Comments
- User Mentions
- Notifications
- Activity Logs

### Dashboard
- Personal Task Statistics
- Project Progress Overview
- Productivity Metrics
- Admin Analytics

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_TaskFlowPro.git
```

### Install Dependencies

```bash
cd backend
npm install
```

### Configure Environment Variables

Create a `.env` file inside the backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### Run Backend

```bash
npm run dev
```

### Run Frontend

Open:

```text
frontend/index.html
```

or use Live Server.
